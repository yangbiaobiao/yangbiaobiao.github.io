git常用命令和本地配置

####git clone 下载线上仓库地址

#####git status 查看当前的仓库状态

#####git add 命令主要用于把我们要提交的文件的信息添加到索引库中。当我们使用git commit时，git将依据索引库中的内容来进行文件的提交。

#####git commit 提交到暂存区

#####git remote add origin git@github.com:yourname/learngit.git

####git push -u origin master
