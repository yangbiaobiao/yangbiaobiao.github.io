伪元素

#伪元素

###first-line:伪元素用于向文本的首行设置特殊样式。例如：


p:first-line
{
color:#ff0000;
font-variant:small-caps;
}


first-letter：伪元素用于向文本的首字母设置样式。例如：



p:first-line
{
color:#ff0000;
font-variant:small-caps;
}
