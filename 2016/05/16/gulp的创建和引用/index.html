gulp的创建和引用

Veröffentlicht am 2016-05-16

1、创建一个项目文件到你所想要盛放gulp的文件中

2、在终端中输入

sudo npm install gulp -g(其中sudo是在mac系统下获取安装权限，而-g则是全局安装)

3、在终端输入npm install –global gulp

4、作为项目的开发依赖（devDependencies）安装,需要在在终端中输入：npm install –save-dev gulp（

其中–save-dev后边跟的为插件的插件名称，安装插件成功后可以在项目初始化的package.json中的devDependencies找到，若没有则表示你所想要安装的插件没有安装成功，具体如图所示：
