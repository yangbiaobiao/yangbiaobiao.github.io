面向对象

this的作用总结




第一种：

全局作用域直接使用指向window

第二种：

普通函数调用 this 指向window,在执行过程应为window.fn

第三种：

作为对象属性调用，指向该对象obj,如以下样式;


        var obj = {
        name:’123’,
        fn3:function () {
        console.log(this.name)
            }
        }
        obj.fn3();

第四种：apply call
apply(op,[1,2,3])apply和call的差别不大，apply只是将参数用数组包括起来，而up.apply则是说明将up的this转向了op

uo.call也是同样的意思；
